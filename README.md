# Emotion-Classification-System
Built an enhanced emotion classification system using deep learning that classifies 6 emotions (anger, fear, joy, love, sadness, surprise) in text. Started with 4 classical ML models (Logistic Regression, SVM, etc.) using TF-IDF vectorization, then implemented an LSTM-based deep learning model with embedding layer. Added 6 novelty features: multi-label classifier (multiple emotions), intensity regressor (emotion strength 0-1), sarcasm detector, context-aware model (with attention mechanism), slang detector, and emoji processor — all deployed on a Flask web app.
